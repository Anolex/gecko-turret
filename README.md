# Mindustry Turrets Extension

This Mindustry mod add a new laser turret to handle the air targets.

## Gecko

Gecko is a compact laser SPAA turret. It targets only air units, consumes only electricity, and fires a continuous laser beam at long range.

Features:

- Electricity turret
- Handles air targets
- No ammo items
- No projectile bullets
- Powerful continuous beam handles the most targets
